1. NavBar


I got my inspiration of my navbar from w3school.com. It is a website for learning various programming language like HTML, CSS, JAVASCRIPT, SQL, PYTHON, etc. I very much liked the simple, clean and modern look of the navbar and i simplified it even more to make it more clean.



2. Home Page


I got my inspiration of my home page from apple.com. It is a website that sells apple products. I very much liked the simple, clean and modern look of the homepage. It looks very clean and i implemented it it a similar way.



3. Store Page


I got my inspiration of my store page from daraz.com. It is a website for an online store. It organizes the products in a grid array which makes the page look organized. I implemented the same grid structure in my website.



4. footer


I got my inspiration of my footer from lays.com. It is a website for lays company. Though the colour of this footer is too bright, when implementing it in my own web page, I made it a bit darker to fit my web page.
